### The 'ticTacToe.sh' script implements the simple TicTacToe game.
It is written in bash.
To play game please execute:
```shell
./ticTacToe.sh
```

To enable/disabled clearing the screen
```shell
#to active
export ACTIVE_CLEAR=1

#to disable
unset ACTIVE_CLEAR 
```

To enable/disabled debug logging
```shell
#to active
export LOG_DEBUG=1

#to disable
unset LOG_DEBUG
```
